# 📊 Superstore Sales Performance Dashboard

## 📌 Project Overview

This project is an interactive Power BI dashboard developed using the Sample Superstore dataset. It provides meaningful insights into sales performance, profitability, customer segments, product categories, and regional performance through interactive visualizations.

---

## 🚀 Features

- KPI Cards
  - Total Sales
  - Total Profit
  - Total Quantity
  - Total Orders

- Interactive Filters
  - Region
  - Segment
  - State
  - Category
  - Order Date

- Visualizations
  - Sales by Category
  - Profit by Region
  - Sales by State (Map)
  - Monthly Sales Trend
  - Sales by Segment

---

## 🛠 Tools Used

- Power BI Desktop
- Microsoft Excel
- DAX (Data Analysis Expressions)
- Power Query

## 📊 Key Insights

- Technology generates the highest sales.
- West region has the highest profit.
- Consumer segment contributes the largest share of sales.
- Monthly sales trend helps identify seasonal patterns.
- Interactive slicers allow users to explore data dynamically.
